O que é o Área 51

Solução licenciada de identidade e acesso para organizações que precisam implantar autenticação federada com entrega documentada — não é software genérico de e-commerce nem plataforma IAM completa pronta para produção sem hardening.

Transparência: o produto é de titularidade do Governo Federal. A Cara Core Informática implementa e presta o Suporte Área 51 sob contrato. O código entrega-se mediante licença de uso — não há download público avulso.

Em uma frase

  • Produto: pacote licenciado com scripts OIDC, ambiente de laboratório Keycloak, testes e documentação.
  • Serviço: equipa Cara Core no ambiente do cliente — integração, acompanhamento e evolução contratada.
  • Loja (este site): vitrine institucional; não executa checkout OAuth — explica e canaliza pedidos.
