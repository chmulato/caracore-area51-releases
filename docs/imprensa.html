🏢 Sobre a Cara Core Informática

A Cara Core Informática é especialista em transformação digital e segurança de identidade federada para órgãos públicos, empresas e integradores.

Missão

Democratizar o acesso a autenticação federada moderna (OIDC/OAuth 2.1) através de soluções open-source, seguras e de fácil implementação.

O Produto: Área 51

Desenvolvimento conjunto entre Cara Core e Governo Federal:

  • Scripts baseline — Configuração OIDC padronizada (Python)
  • Deploy automatizado — Keycloak em Docker com um comando
  • Segurança integrada — PKCE, JWT, OAuth 2.1
  • Documentação completa — Setup, conceitos, troubleshooting

📋 Casos de Uso Reais

1. Órgão Federal — Proteger Portal Interno

Desafio: Portal interno sem proteção adequada de acesso

Solução: Area 51 + Keycloak + Integração Entra ID

Resultado: Implementado em 3 dias, acesso federado seguro, PKCE validado

Impacto: Segurança 100%, conformidade NIST 800-63

2. Empresa Média — Oficina de OIDC

Solução: Ambiente de treinamento com Area 51

Impacto: Equipe capacitada em segurança de identidade
